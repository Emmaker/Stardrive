# Contributing

Contributions to the project may be submitted on [GitHub](https://github.com/Emmaker/Stardrive), [Forgejo](https://git.emmaker.gay/emmaker/Stardrive), or by email at emmaker@myyahoo.com.
Contributions submitted via GitHub Pull Request will always be closed: this does not mean it is **rejected**, instead Pull Requests will be manually merged and pushed if accepted.
